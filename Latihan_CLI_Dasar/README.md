# CLI DASAR

## Membuat folder "Latihan_CLI_Dasar"
### Command : mkdir Latihan_CLI_Dasar

 ## Membuat file kosong "latihanX.txt" didalamnya sebanyak 5 (X diganti 1-5)
### Command : touch latihan{1..5}

## Melihat daftar file yang sudah di buat
### Command : ls

## Menghapus file dengan urutan ke 4 
### Command : rm latihan4.txt

 ## Membuat folder dengan nama "latihan4.txt"
### Command : mkdir latihan4.txt

## Menghapus file dengan urutan ke 5 
### Command : rm latihan5.txt

## Membuat folder dengan nama "latihan5.txt"
### Command : mkdir latihan5.txt

## Melihat daftar file dan folder yang sudah di buat
### Command : ls

## Menghapus folder dengan nama "latihan4.txt"
### Command : rmdir latihan4.txt
